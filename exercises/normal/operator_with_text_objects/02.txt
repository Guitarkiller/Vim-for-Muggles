Format a paragraph
==================


Mission
-------

Format a paragraph so the lines have a maximum length of 80 characters.


Instructions
------------

  1. Set the text width to 80 characters. `:set textwidth=80`
  2. Place the cursor in the first paragraph.
  3. Combine the format command `gq` with the a-paragraph text object `ap`.
  4. Go to the next paragraph and repeat.
