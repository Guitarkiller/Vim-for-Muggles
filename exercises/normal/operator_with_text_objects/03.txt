Delete inside an HTML tag
=========================


Mission
-------

Remove all the content from the <body> tag.


Instructions
------------

  1. Place the cursor above the tag <body> or </body>.
  2. Combine the deletion operator `d` with the inner-tag text object `it`.


Text
----

<html>
  <head>
    <title>My Webpage</title>
  </head>
  <body>
    <h1>My Webpage</h1>
    <p>Lorem ipsum...</p>
    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
    </ul>
  </body>
</html>
